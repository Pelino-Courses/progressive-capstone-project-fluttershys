# Mini-Capstone Assignment One — Part A–D

## Task Summary
This repository is the SukaApp mini-capstone project for the first integration milestone. The submission demonstrates:

- Part A: Dart fundamentals (variables, null safety, collections, functions)
- Part B: OOP & data models (classes, inheritance, mixins, async/await)
- Part C: Flutter widgets & UI (layouts, custom widgets, Material Design 3 theme)
- Part D: Navigation & forms (named routes, data passing, form validation)

## Submission Requirements
- One YouTube video (8–12 minutes, all members on camera/mic)
- GitHub repo link for this project
- Git tag: `mini-capstone-part-a-d`
- Submit via the Teams assignment by the deadline

## Project Artifacts
- `docs/DESIGN.md` — screen mockups and UI structure for the mini-capstone
- `wireframes/sukaapp_wireframes.html` — interactive wireframe prototypes
